第一步：拷贝下列代码到你的html文件的<head>和</head>之间：

<style type="text/css">
<!--
body {
background-image: url(lvbgcolor.gif);
}
-->
</style>

说明：

　　background-position是设置背景的位置，有两个参数，第一个用于定位前景的上下位置，可以是top、center和bottom，第二个用于定位前景的左右位置，可以是left、center和right，参数也可以是具体的值，如background-position:100px 200px。background-repeat是设置背景的表现方式，有repeat、no-repeat、repeat-x和repeat-y四种，repeat就是背景可以在x和y两个方向上重复平铺显示，no-repeat则是只显示一个背景，repeat-x和repeat-y分别是在x和y方向上重复平铺显示。
